# üéì Students Performance Insights Dashboard

An interactive Power BI dashboard providing deep insights into students' academic performance across math, reading, and writing.  
The dashboard highlights performance gaps, demographic influences, and the effectiveness of test preparation programs.

---

##  Overview
This project analyzes the **Students Performance** dataset and visualizes key metrics using a fully interactive Power BI dashboard.  
Users can filter results by gender, race/ethnicity, parental education level, lunch type, test preparation course, and score range.

---

##  Key Insights
